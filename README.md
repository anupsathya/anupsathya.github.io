# Personal Website

A lightweight personal website built with 11ty and styled with Water.css.

## Features

- Minimal and fast
- Responsive design
- Blog-ready
- Easy to customize

## Getting Started

1. Clone this repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. Build for production:
   ```bash
   npm run build
   ```

## Structure

- `src/`: Source files
  - `_layouts/`: Layout templates
  - `index.md`: Homepage
  - `about.md`: About page
- `css/`: Custom styles
- `.eleventy.js`: 11ty configuration
- `_site/`: Generated site (after building)

## Customization

1. Modify the content in `src/` directory
2. Update styles in `css/custom.css`
3. Add new layouts in `src/_layouts/`
4. Configure 11ty settings in `.eleventy.js`
